    📨 MQ Topic 管理

    • /mq-list [关键词] [env] 列出 Topic (支持模糊搜索)
    • /mq-exists <topic> [env] 检查 Topic 是否存在
    • /mq-last <topic> [env] 最新一条完整消息
    • /mq-create <topic> [env] 创建 Topic
    • /mq-delete <topic> [env] 删除 Topic (需确认)
    • /mq-send <topic> <json> [env] 发送 JSON 消息
    • /mq-recent <topic> [分钟] [env] 最近 N 分钟消息
    • /mq-query <topic> <字段=值> [分钟] [env] 条件查询消息
    环境: dev, test, uat (默认 test) · 环境参数放最后
